# NutriSnap

Aplicativo de análise nutricional com IA

## 📱 Sobre o Projeto
NutriSnap é um app que permite analisar alimentos através de fotos, utilizando inteligência artificial para fornecer informações nutricionais.

## 🏗️ Estrutura
- `/nutrisnap` - Backend em Django
- `/nutrisnap_app` - Frontend em Flutter
